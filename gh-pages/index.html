SVG Invader Sprite Generator
  • GitHub

Description

This module generates SVG sprites that look similar to Space Invaders era sprites. It can be used as a command line application to generate sprites, or loaded as a node module in Node.js Javascript or Typescript apps, or with webpack/browserify/etc. in the browser.

Node.js Usage

You can view the README on GitHub for command line and Node.js (Javascript and Typescript) documentation.
